# 糖糖 APP 项目说明书

## 1. 项目概述

### 1.1 项目名称
**糖糖 (TangTang)** - 饮食记录与碳水计算辅助工具

### 1.2 项目定位
糖糖是一款面向糖尿病及血糖管理人群的移动应用，通过智能化的饮食记录和碳水计算功能，帮助用户轻松管理日常饮食，稳定血糖水平。

### 1.3 核心价值
- **便捷记录**：支持拍照、语音、文字三种输入方式
- **智能分析**：自动识别食物并计算营养成分
- **健康激励**：通过健康树、勋章系统等游戏化元素激励用户坚持记录
- **数据可视化**：提供碳水和血糖趋势图表，直观展示健康进展

## 2. 功能模块详解

### 2.1 首页（记录）

#### 2.1.1 主要功能
| 功能 | 描述 |
|------|------|
| 今日碳水概览 | 显示当日已摄入碳水总量、目标值及进度条 |
| 餐后血糖预测 | 基于当前碳水摄入预测餐后血糖范围 |
| 快捷输入 | 拍照识别、语音输入、文字输入三种方式 |
| 快速复记 | 展示今日最近 3 条记录，支持一键复记 |
| 控糖挑战 | 7 天连续记录挑战，可视化进度展示 |
| 健康树 | 右侧悬浮 widget，展示连续记录天数和等级 |
| 勋章系统 | 3 天/7 天/30 天连续记录勋章 |

#### 2.1.2 交互设计
- **拍照输入**：调用摄像头或相册，模拟 AI 识别食物
- **语音输入**：调用 Web Speech API 进行语音识别
- **文字输入**：手动输入食物名称和重量

### 2.2 结果页

#### 2.2.1 主要功能
| 功能 | 描述 |
|------|------|
| 食物识别结果 | 显示识别出的食物名称、图片 emoji、置信度 |
| 营养数据 | 碳水、热量、蛋白质、脂肪含量 |
| 数据纠错 | 支持重新拍照、搜索食物、手动修改名称 |
| 饮食建议 | 根据血糖值生成个性化饮食建议 |
| 餐后血糖记录 | 记录餐后 2 小时血糖值 |

#### 2.2.2 食物红绿灯分类
- **绿灯食物**：碳水极低，可以放心食用（如：鱼肉、鸡蛋、蔬菜）
- **黄灯食物**：碳水适中，注意份量控制（如：米饭、水果、面条）
- **红灯食物**：碳水/脂肪较高，建议减少食用（如：红烧肉、炸鸡）

#### 2.3.1 时间视图切换

## 3. 技术架构

### 3.1 技术栈
| 类别 | 技术 |
|------|------|
| 前端框架 | 原生 HTML5 + CSS3 + JavaScript |

| 摄像头访问 | navigator.mediaDevices.getUserMedia |

### 3.2 数据模型

#### 3.2.1 记录数据 (Record)
```javascript
{
  id: number,              // 唯一标识
  timestamp: Date,         // 记录时间
  name: string,            // 食物名称
  carbs: number,           // 碳水化合物 (克)
  calories: number,        // 热量 (千卡)
  protein: number,         // 蛋白质 (克)
  fat: number,             // 脂肪 (克)
  image: string,           // 食物 emoji
  confidence: string,      // 置信度 (高/中/低/手动)
  advice: string,          // 饮食建议
  postMealBG: number|null, // 餐后血糖 (mmol/L)
  preMealBG: number|null   // 餐前血糖 (mmol/L)
}
```

#### 3.2.2 用户数据 (UserData)
```javascript
{
  streak: number,                    // 连续记录天数
  lastRecordDate: string|null,       // 最后记录日期
  badges: {                          // 勋章状态
    '3days': boolean,
    '7days': boolean,
    '30days': boolean
  },
  challenge: {                       // 挑战进度
    startDate: string|null,
    completedDays: string[]
  },
  postMealBG: number[]               // 餐后血糖记录
}
```

### 3.3 本地存储

| `tangtang_userdata` | 用户数据对象 |

## 4. 核心算法

### 4.1 餐后血糖预测
```javascript
预测血糖 = 空腹血糖 (5.5) + 碳水摄入量 × 0.04
```

### 4.2 连续记录天数计算
- 今天已记录：保持当前天数
- 昨天记录过：天数 +1
- 更早记录：重置为 1

### 4.3 健康树等级系统
| 连续天数 | 等级 | Emoji |
|----------|------|-------|
| 1-4 天 | 萌芽 | 🌱 |
| 5-9 天 | 成长 | 🌿 |
| 10-19 天 | 茂盛 | 🌲 |
| 20-29 天 | 参天大树 | 🌳 |
| 30 天 + | 结果 | 🍎 |

### 4.4 控糖评分算法
- **碳水控制分**：(目标内摄入 / 总摄入) × 100
- **血糖稳定分**：基于血糖波动范围评估
- **记录习惯分**：基于记录频率评估
